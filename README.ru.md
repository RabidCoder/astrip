# AStrip

AStrip — лёгкая CLI-утилита для предобработки текста перед импортом в Anki.

Инструмент предоставляет безопасный и детерминированный пайплайн для очистки исходных текстовых данных.

## Пайплайн

### 1. Prepare (подготовка)
- Создаёт входной и выходной рабочие файлы
- Проверяет принадлежность файлов системе через служебный маркер
- Защищает от обработки посторонних файлов
- Обнуляет рабочее окружение для новой сессии

### 2. Process (обработка)
- Читает входной файл построчно
- Нормализует текст:
  - приведение к нижнему регистру
  - очистка пробелов и whitespace (пробелы, табы, переносы строк)
- Удаляет пустые строки
- Удаляет дубликаты (после нормализации)
- Записывает очищенный результат в выходной файл

---

## 🚀 Быстрый старт

1. Запустить этап prepare  
2. Добавить исходный текст во входной файл  
3. Запустить этап process  
4. Получить очищенный результат  

---

## Модель безопасности

- Входной файл никогда не изменяется во время обработки
- Файлы проверяются через маркер системы
- Посторонние файлы не обрабатываются
